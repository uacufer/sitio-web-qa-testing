Bienvenido a QA Testing

Este sitio web es un ejemplo práctico de testing integrado donde QA valida HTML semántico, CSS responsive, y usa Git para colaboración profesional.

Features Principales

HTML Semántico

Estructura semántica correcta con etiquetas HTML5 apropiadas para accesibilidad.

CSS Responsive

Diseño responsive que adapta a móvil (320px), tablet (768px) y desktop (1024px+).

Git Workflow

Versionado en Git con branching strategy, PRs y code review para QA.

Accesibilidad

Diseño accesible con contraste WCAG AA, focus indicators y navegación keyboard.

Performance

CSS optimizado, sin código muerto, animaciones eficientes en GPU.

Compatibilidad

Testeado en Chrome, Firefox, Safari y Edge con prefijos vendor donde necesario.

Sobre Este Proyecto

Este sitio web es un proyecto educativo para aprender testing QA profesional. Incluye validación HTML, pruebas CSS, control de versiones Git y colaboración mediante Pull Requests.

Tecnologías Usadas

  • HTML5 semántico
  • CSS3 responsive con media queries
  • JavaScript vanilla para validación
  • Git para control de versiones
  • GitHub para colaboración
